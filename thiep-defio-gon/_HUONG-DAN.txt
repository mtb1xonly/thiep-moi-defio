========================================================
THIỆP MỜI DEFIO MEDIANET - LUỒNG GỌN (1 template + quản lý link)
========================================================

Gói này chỉ có 2 file chính:

  • thiep.html   = TEMPLATE thiệp (1 file dùng cho TẤT CẢ khách)
  • quanly.html  = TRANG QUẢN LÝ (bạn dùng để copy link mời từng khách)

--------------------------------------------------------
CÁCH HOẠT ĐỘNG
--------------------------------------------------------
Tên khách được truyền qua đường link, ví dụ:
  thiep.html?ten=Mr Trần Nhật Quang
File thiep.html đọc tên đó rồi hiện lên thiệp. Một file phục vụ cả 27 khách.

--------------------------------------------------------
BƯỚC 1 — UP LÊN VERCEL (qua GitHub)
--------------------------------------------------------
1. Tạo repo GitHub mới (Public), ví dụ tên "thiep-defio".
2. Upload 2 file: thiep.html và quanly.html lên repo.
3. Vào vercel.com > Add New Project > chọn repo đó > Deploy.
4. Vercel cho địa chỉ kiểu: https://thiep-defio.vercel.app

--------------------------------------------------------
BƯỚC 2 — LẤY LINK MỜI CHO TỪNG KHÁCH
--------------------------------------------------------
1. Mở: https://thiep-defio.vercel.app/quanly.html
2. Ở ô trên cùng, dán địa chỉ template:
     https://thiep-defio.vercel.app/thiep.html
   (trang sẽ nhớ, lần sau khỏi nhập lại)
3. Mỗi khách có nút "Copy link". Bấm là link mời riêng vào bộ nhớ tạm.
4. Dán vào Zalo gửi khách. Xong.

Nút "Mở thử" để bạn xem trước thiệp của khách đó.
Nút "Copy toàn bộ" để lấy 1 lần cả 27 link kèm tên.

--------------------------------------------------------
LƯU Ý
--------------------------------------------------------
• Phải chạy trên web thật (sau khi up Vercel) thì link ?ten= mới hoạt
  động chuẩn. Mở file trực tiếp từ máy có thể không nhận tên.
• Muốn thêm/bớt/sửa tên khách: sửa danh sách trong quanly.html
  (phần const GUESTS = [...]) hoặc nhờ Claude sửa.
• Phần xác nhận tham dự HIỆN CHƯA gắn backend Google Sheet. Khi nào
  bạn dựng xong Sheet và gửi link backend cho Claude, sẽ gắn vào thiep.html
  thì dữ liệu mới về Sheet.
